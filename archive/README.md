# Archive

Source material that is **not** used by the live site but is kept for provenance
and possible future use. Nothing here is loaded by any page.

## Files

- **`board_archive_pointers.xlsx`** — a curated index of ~25 music education
  journals' editorial-board pages and their Wayback Machine snapshot URLs, plus a
  "How to use" guide. This is the *source-hunting* layer behind the editorial
  histories: it stores links (not transcribed names) so it never goes stale.
  Use it to find archived board pages when extending or verifying
  `data/xlsx/editorial_histories.xlsx`.

> Removed from active data on 2026-06-22 because no page references it. Retained
> here rather than deleted, as the pointers are unique research data not duplicated
> elsewhere.
